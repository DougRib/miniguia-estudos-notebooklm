# Imagens do projeto

Adicione nesta pasta prints do seu caderno no NotebookLM.

## Sugestões de prints

- `notebooklm-fontes.png` — mostrando as fontes adicionadas.
- `notebooklm-prompts.png` — mostrando os prompts testados.
- `notebooklm-resumo.png` — mostrando um resumo gerado.
- `notebooklm-glossario.png` — mostrando o glossário.
- `notebooklm-miniguia.png` — mostrando o miniguia final.

Essas imagens deixam o repositório mais visual e ajudam a comprovar o processo de criação.
